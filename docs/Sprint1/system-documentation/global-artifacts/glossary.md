# Glossary

**Terms, Expressions and Acronyms (TEA) must be organized alphabetically.**

| **_TEA_** (PT) | **_Descrição_** (PT) |
|:----------------|:-------------------------------|
| **Administrador DGS** | Elemento da Direcção-Geral da Saúde responsável pela administração do sistema PVMS; gere centros de vacinação, vacinas e pessoal. |
| **Agendamento** | Pedido de marcação de vacinação efectuado por um utente, indicando centro, data, hora e tipo de vacina pretendida. |
| **Capacidade** | Número máximo de utentes que podem ser vacinados num determinado centro e período de tempo. |
| **Centro de Vacinação** | Local onde são administradas vacinas. Existem centros de saúde (fixos) e centros de vacinação em massa (temporários). |
| **Certificado de Vacinação** | Documento emitido após completar o esquema vacinal, comprovando as vacinas administradas e as respectivas datas. |
| **DGS** | Direcção-Geral da Saúde, entidade nacional responsável pelo sistema. |
| **Doença-Alvo** | Doença ou agente patogénico contra o qual uma vacina proporciona imunização (ex.: COVID-19, gripe). |
| **Enfermeiro** | Profissional responsável por administrar vacinas e registar o processo, incluindo lote, marca e reacções adversas. |
| **Marca de Vacina** | Designação comercial de uma vacina fornecida por um fabricante específico. |
| **Pessoal** | Conjunto de funcionários (rececionistas e enfermeiros) que trabalham nos centros. |
| **Processo de Vacinação** | Registo criado quando a vacina é administrada, contendo data/hora, utente, enfermeiro, tipo de vacina, marca, lote e observações. |
| **Rececionista** | Funcionário que confirma chegada do utente e actualiza o estado do agendamento. |
| **Reacção Adversa** | Efeito secundário ou resposta inesperada apresentada por um utente após vacinação. |
| **SNS** | Sistema Nacional de Saúde. Cada utente é identificado por um número único de SNS. |
| **Tipo de Vacina** | Classificação segundo a tecnologia utilizada (ARNm, vector viral, subunidade proteica, etc.). |
| **Utente** | Pessoa identificada pelo número de SNS que pode agendar e receber vacinas. Inclui dados pessoais e de contacto. |
| **Vacina** | Produto biológico destinado a induzir imunidade contra uma determinada doença. No PVMS, caracteriza-se por código, doença-alvo, descrição, tipo, marca, lote e validade. |
